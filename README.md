Idle callback → buat square tengah tu berpusing non-stop.
Mouse left click/drag → lukis square kecik warna random.
Mouse move tanpa klik → player square warna biru ikut cursor.
Arrow keys → gerakkan player square macam game.
Key P → pause / resume animation (tunjuk idle boleh control).
Key C → clear semua square yang dilukis.
Right click / Q / ESC → quit program.
Resize window → scene still cantik sebab reshape callback update gluOrtho2D.
